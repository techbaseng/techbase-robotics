About This Course

The complete micro:bit catalogue.

This course contains all 123 official projects from the BBC micro:bit foundation at microbit.org — the complete Make It: Code It catalogue. Every single project is included, arranged in order from absolute beginner to advanced.

Each project page on this site links directly to the official microbit.org guide where you will find the full step-by-step instructions, MakeCode starter code, and extension challenges. You only need a BBC micro:bit v2, a USB cable, and a Chrome or Edge browser to get started. Designed and compiled by Babatunde Awoyemi, Techbase Consultant Services, Ibadan.

How It Works

Learn at your own pace.

01

Get a micro:bit

BBC micro:bit v2 is widely available from electronics suppliers. A USB cable and Chrome browser are all you need to start.

02

Pick a Project

Browse the 123 projects above. Start at 001 or jump to any level. Each card shows the title and a short description.

03

Open micro:bit.org

Every project page has a red button that opens the official guide and MakeCode editor directly on microbit.org.

04

Build & Extend

Follow the official guide, upload to your micro:bit, and try the extension challenges at the end of each project.
