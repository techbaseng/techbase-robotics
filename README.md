## About This Course

Robotics is where code meets the physical world. In this course, you will learn how to program microcontrollers using Arduino — building circuits, reading sensors, controlling motors, and ultimately assembling your own autonomous robot.

This is a hands-on, practical course. Every lesson involves real hardware, real wiring, and real code running on a physical board. By the end, you will have built a robot that can sense its environment and respond to it automatically.

**Some experience with programming (Scratch or Python) is recommended before starting this course.**

---

## What You Will Learn

By the end of this course you will be able to:

- Explain what a microcontroller is and how Arduino works
- Set up the Arduino IDE and upload code to a board
- Use digital and analog pins to control LEDs and read sensors
- Control servo and DC motors
- Read input from ultrasonic, infrared, and temperature sensors
- Use serial communication to debug your projects
- Wire a basic circuit on a breadboard safely
- Build and program a line-following or obstacle-avoiding robot

## How to Use This Course

1. **Start here** — read this page and gather your components (see list below)
2. **Open Lesson 01** — click the link in the table above
3. **Read the lesson** — each file explains the concept with circuit diagrams and code
4. **Build and upload** — wire the circuit and upload the code to your Arduino
5. **Move forward** — go to the next lesson whenever you are ready

> ⚠️ **Safety note:** Always disconnect your Arduino from power before changing your circuit wiring. Double-check connections before powering on.

## Components You Will Need

| Component | Quantity | Purpose |
|-----------|----------|---------|
| Arduino Uno (or Nano) | 1 | Main microcontroller board |
| Breadboard | 1 | Connect components without soldering |
| Jumper wires (M-M, M-F) | 30+ | Make circuit connections |
| LEDs (assorted colours) | 10 | Output indicator exercises |
| 220Ω resistors | 10 | Protect LEDs from too much current |
| Push buttons | 4 | Digital input exercises |
| Potentiometer (10kΩ) | 1 | Analog input exercise |
| HC-SR04 ultrasonic sensor | 1 | Distance measurement |
| IR sensors | 2 | Line following / obstacle detection |
| Servo motor (SG90) | 1 | Motor control exercises |
| DC motors + wheels | 2 | Robot chassis drive |
| L298N motor driver | 1 | Control DC motors from Arduino |
| USB cable (Type A to B) | 1 | Connect Arduino to computer |

> Most of these components are available at electronics markets in Ibadan and Lagos, or from online suppliers in Nigeria.
