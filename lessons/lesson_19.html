Simple Door Alarm

Build a real door alarm that detects when a door is opened

🔴  Open on micro:bit.org  ↗ Opens the official project page with starter code and the MakeCode editor

What You Will Build

Build a real door alarm that detects when a door is opened. Use the step-by-step guide below alongside the official micro:bit project page — the button above opens the MakeCode starter file directly.

What You Will Learn

  • Connect conductive foil strips to micro:bit pins using crocodile clips
  • Use digital pin reading to detect when a circuit is open or closed
  • Build a practical real-world security device
  • Implement arming and disarming logic using a boolean variable

Step-by-Step Guide

  • 1Cut two strips of aluminium foil and stick one to the door edge and one to the door frame so they touch when the door is closed.
  • 2Connect one crocodile clip from the foil strip on the door to micro:bit Pin 0.
  • 3Connect a second clip from the door-frame foil strip to micro:bit GND.
  • 4Open MakeCode and add a forever block inside it.
  • 5Read digital read pin P0 — when the foil strips touch the circuit is closed and it reads 0. When the door opens and strips separate it reads 1.
  • 6Add a condition: if digital read P0 = 1 then play an alarm sound and flash the display repeatedly.
  • 7Add a Button A press that toggles an armed boolean variable between true and false.
  • 8Only trigger the alarm inside the condition when armed is true.
  • 9Add a 10-second countdown when arming so you have time to leave the room after pressing A.
  • 10Test carefully by gently opening and closing the door and observing the alarm response.

Extension Challenges

Completed the main project? Push further with these challenges:

  • Add Radio to send an immediate alert to a second micro:bit placed in another room.
  • Add a secret PIN entry system: press A and B in a specific sequence to silently disarm the alarm.
  • Add a counter that tallies how many times the door was opened during a monitoring session.
