Hot and Cold Game

Play a two-player searching game using radio signal strength as a proximity clue

🔴  Open on micro:bit.org  ↗ Opens the official project page with starter code and the MakeCode editor

What You Will Build

Play a two-player searching game using radio signal strength as a proximity clue. Use the step-by-step guide below alongside the official micro:bit project page — the button above opens the MakeCode starter file directly.

What You Will Learn

  • Use radio communication to exchange signals between two micro:bits
  • Use received signal strength as the core interactive game mechanic
  • Design a fair and fun two-player game with clear feedback
  • Understand what RSSI (received signal strength indicator) means

Step-by-Step Guide

  • 1Open MakeCode — you will write two separate programs: one for the HIDER and one for the SEEKER.
  • 2HIDER program: set radio group 5. In the forever block, transmit the number 1 every 300ms.
  • 3SEEKER program: set radio group 5. On received number, read and store the current signal strength.
  • 4Map the signal to temperature-style feedback: strength > -50 → 'HOT!' with flame icon; > -70 → 'WARM'; > -90 → 'COOL'; anything weaker → 'COLD!' with snowflake icon.
  • 5Display both the text word AND the matching icon together for the clearest possible feedback.
  • 6Add audio: HOT plays a rapid high-pitched beep; COLD plays a single slow low-pitched beep.
  • 7Make the display update faster when the signal is stronger — use shorter pause values based on the zone.
  • 8Download the HIDER program to one micro:bit and the SEEKER program to the other.
  • 9Give the HIDER micro:bit to one player to hide anywhere in the room while the other player closes their eyes.
  • 10The SEEKER's display guides the searching player — brighter and louder means they are getting closer!

Extension Challenges

Completed the main project? Push further with these challenges:

  • Add a freeze mode where the Seeker gets no feedback for 5 random seconds to make the game harder.
  • Make it a team game with multiple HIDER micro:bits — the Seeker must find all of them to win.
  • Add directional hints: the Hider player can press A or B to send a Left or Right hint via Radio.
